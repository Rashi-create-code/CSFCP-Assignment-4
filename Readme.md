# **Project Title**

**Python Calculator**

---

# **Description**

A simple command-line calculator built in Python that performs addition, subtraction, multiplication, and division. Created using VS Code, Git, and GitHub as part of Assignment 4.

---

# **How to Run**

1. Open the project folder in VS Code.
2. Run the command:

```
python calculator.py
```

3. Enter numbers and an operator as instructed.

---

# **Features**

* Add, subtract, multiply, divide
* Input validation
* Operator validation
* Division-by-zero handling
* Clear output formatting
* Loop to perform multiple calculations
